# Практична робота №3: Каркас та стилістика "Study Buddy"

## 1. Головний сценарій ("Щасливий шлях")
Студент заходить на головну сторінку → Бачить кнопку "Знайти напарника" або "Створити запит" → Натискає її → Заповнює коротку форму (предмет, час) → Натискає "Опублікувати" → Отримує підтвердження, що запит з’явився у стрічці.

**Альтернативні стани:**
- Порожня стрічка: повідомлення "Запитів поки немає, будь першим!" + велика кнопка дії.
- Помилка: якщо поле "Предмет" порожнє, кнопка публікації неактивна або з’являється червона підказка.

---

## 2. Візуальна ієрархія та сітка
- Сітка: 8-піксельна (відступи 8, 16, 24, 32 px).
- Вирівнювання: заголовки, картки та кнопки по лівому краю.
- Акцент: кнопка "Створити запит" яскравим кольором у правому верхньому куті та дубльована по центру при порожній стрічці.

---

## 3. Стильова палітра та типографіка
| Елемент        | Значення / Колір | Призначення |
|----------------|------------------|-------------|
| Primary Color  | #4A90E2 (Синій)  | Головна кнопка дії, активні фільтри |
| Background     | #F5F7FA (Світло-сірий) | Фон сторінки |
| Text Primary   | #1A202C (Темно-синій) | Заголовки та основний текст |
| Error Color    | #E53E3E (Червоний) | Повідомлення про помилки |
| Шрифт          | Inter / System UI | Без засічок, читабельний |

---

## 4. Структура головної сторінки (Вайрфрейм)
- **Header (Шапка):** Логотип зліва, пошук по центру, кнопка "+ Створити запит" справа.  
- **Hero Section:** Заголовок "Знайди напарника для навчання сьогодні" + опис цінності.  
- **Filter Bar:** Фільтри за предметами (Математика, Програмування, Англійська).  
- **Content Zone:** Стрічка карток (Ім’я, Предмет, Час, опис, кнопка "Відгукнутися").  

---

## 5. Доступність (Accessibility)
- Контраст тексту до фону ≥ 4.5:1 (WCAG).  
- Мінімальний розмір кнопок: 44x44 px.  
- Помилки супроводжуються кольором + іконкою/текстом.
